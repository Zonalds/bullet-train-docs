Overview

Bullet Train lets you manage features across web, mobile and server side applications. Bullet Train is 100% Open Source. Host yourself or let us take care of the hosting.

The application consist of 3 components:

  1. The Server-Side REST API.
  2. The Front End Administration Web Interface.
  3. Client Libraries.

To get up and running, you can either use https://bullet-train.io/ for 1 and 2 above, or you can self host the API and Front End. Once you have these components up and running, you can add the client libraries to your apps and start managing your features remotely.
